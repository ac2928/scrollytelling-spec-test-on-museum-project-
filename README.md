# Museum Project — The Evolution of Web Design

## Live Site

## Project Overview
This project is a digital museum experience that explores the evolution of web design from the 1990s to the present.

The goal is not just to present information, but to guide the visitor through a structured, scroll-based narrative that shows how web design developed from technical limitation into a user-centered discipline.

The site is designed as a guided exhibit, where each section builds on the previous one to create a clear and intentional learning experience.

---

## Core Narrative
The central argument of this museum is that web design evolved from solving technical constraints to shaping human experience.

The visitor moves through a guided timeline:

- Early Web (1990s): functionality over design  
- Web 2.0 (2000s): interaction and engagement  
- Modern Web: user experience and responsiveness  
- Future: AI-driven and personalized design  

The final section, **“Why It Matters,”** connects this evolution to real-world impact and user behavior, reinforcing the educational purpose of the exhibit.

---

## Research Foundation
Before building the site, research was conducted to identify:

- key stages in the evolution of web design  
- major technological and conceptual shifts  
- how user experience became central over time  

This research informed the structure, ensuring the site presents a meaningful and accurate narrative rather than a random collection of facts.

---

## Design Framework

### Design Style — Swiss Minimal
The site uses a Swiss design approach to ensure clarity, structure, and consistency.

This is reflected in:

- Clean grid-based layout  
- Consistent spacing and alignment  
- Minimal color palette (black, white, gray)  
- Clear typographic hierarchy  

The goal was to remove visual noise and allow the content and narrative to guide the experience.

---

### Brand Archetype — Sage
The site follows the **Sage archetype**, focusing on education, clarity, and insight.

This influences:

- Informational and educational tone  
- Clear explanations  
- Structured content delivery  

The experience is designed to feel like a curated museum exhibit that teaches the visitor something meaningful.

---

### Cialdini Principle — Authority
The principle of **Authority** is used to increase credibility and engagement.

This is demonstrated through:

- Structured, confident explanations  
- Organized presentation of information  
- A museum-style format that reinforces trust  

This encourages the user to continue exploring the exhibit.

## AI Orchestration Process

This project was built using a structured AI workflow rather than a single prompt.

### Phase 1 — Planning
- Defined the topic and core narrative before building  
- Identified key sections and user flow  
- Studied reference design systems and museum examples  

### Phase 2 — Development
- Used AI to generate layout ideas and structure  
- Built the site using HTML and CSS  
- Applied a consistent design system  

### Phase 3 — Iteration and Refinement
- Fixed image loading issues using reliable sources  
- Adjusted layout spacing for better visual balance  
- Improved readability and section flow  
- Implemented a scroll-based navigation bar that changes color based on background  

### Phase 4 — Evaluation
- Reviewed the site for clarity, consistency, and usability  
- Ensured alignment with Swiss design principles  
- Refined content to strengthen narrative flow  

AI was used as a tool for refinement and problem-solving, but all final decisions were made intentionally.

## Reflection

This project demonstrated the importance of planning, structure, and iteration in design.

Instead of building randomly, I followed a spec-driven approach where each decision was guided by a clear goal and narrative.

Breaking the project into stages made it easier to identify problems and improve the site step by step.

One key takeaway was that AI is most effective when used in a controlled workflow. Rather than relying on a single output, I used AI to refine layout, fix issues, and improve design quality over time.

Overall, this project shows how intentional design and structured development can produce a clear, engaging, and effective digital experience.
